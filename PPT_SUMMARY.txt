반도체 불량 패턴 분류 시스템 제작 과정

1. 데이터셋 구축: 8가지 불량 패턴(박스 라벨링 유무 혼합) 총 797개 데이터셋 구축 및 이미지→wafer-map 변환 파이프라인 개발

2. 모델 학습: Focal Loss, Class Weight, Mixup/CutMix 증강 기법 적용하여 CNN 기반 패턴 분류 모델 학습 (검증 정확도 89.47%, 테스트 정확도 83.96%)

3. 추론 파이프라인: YOLO 검출 + wafer-map 생성 + 패턴 분류 2단계 구조로, 검출이 적을 경우 이미지 직접 변환하는 자동 판단 로직 구현

4. GUI 개발: PyQt5 기반 통합 애플리케이션으로 이미지 분석, 불량 검출, 패턴 분류, 결과 시각화를 한 화면에서 제공

5. 성능: 8개 패턴 중 5개 클래스 80% 이상, 3개 클래스 100% 정확도 달성, none/near-full은 박스 라벨링 없이도 정확하게 분류 가능
