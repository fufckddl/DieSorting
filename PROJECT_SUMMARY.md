# 반도체 불량 패턴 분류 시스템 제작 과정

## 프로젝트 개요
본 프로젝트는 반도체 웨이퍼 이미지에서 불량 패턴을 자동으로 분류하는 딥러닝 기반 시스템을 개발하였습니다. YOLO 객체 검출 모델과 CNN 기반 패턴 분류 모델을 결합한 2단계 파이프라인을 구축하여, 8가지 불량 패턴(local, center, edge-local, donut, edge-ring, scratch, near-full, none)을 평균 83.96%의 정확도로 분류할 수 있습니다.

## 주요 개발 내용
1. **데이터셋 구축**: 박스 라벨링이 있는 6개 패턴과 박스 라벨링이 없는 2개 패턴(near-full, none)을 포함한 총 797개의 학습 데이터셋을 구축하고, 이미지를 직접 wafer-map으로 변환하는 통합 데이터 준비 파이프라인을 개발하였습니다.

2. **모델 학습**: 클래스 불균형 문제를 해결하기 위해 Focal Loss와 Class Weight를 적용하고, Mixup/CutMix 증강 기법과 개선된 CNN 아키텍처를 사용하여 패턴 분류 모델을 학습하였습니다. 최종적으로 검증 정확도 89.47%, 테스트 정확도 83.96%를 달성하였습니다.

3. **추론 파이프라인 최적화**: YOLO 검출 결과가 없거나 적은 경우(none/near-full) 이미지를 직접 wafer-map으로 변환하는 자동 판단 로직을 구현하여, 박스 라벨링이 없는 패턴도 정확하게 분류할 수 있도록 개선하였습니다.

4. **GUI 애플리케이션 개발**: PyQt5 기반의 사용자 친화적인 GUI를 개발하여, 이미지 업로드부터 불량 검출, 패턴 분류, 결과 시각화까지 전체 프로세스를 통합적으로 제공합니다. 모든 패턴의 확률 분포를 실시간으로 확인할 수 있습니다.
